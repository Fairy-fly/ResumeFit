# Screenshots

该目录用于放置 GitHub README、项目报告或简历展示中使用的截图。当前仅提供截图清单，请在本地运行项目后手动补充图片。

| 截图名称 | 建议文件名 | 截图重点 |
| --- | --- | --- |
| 首页 / Dashboard | `dashboard.png` | 展示完整流程、完成进度和下一步建议 |
| 简历输入页 | `resume-input.png` | 展示通用简历输入和历史详情 |
| 项目库页面 | `project-library.png` | 展示项目经历、技术栈和个人贡献 |
| JD 分析页 | `jd-analysis.png` | 展示 JD 原文和结构化岗位分析 |
| 匹配度报告页 | `match-report.png` | 展示匹配分数、优势、差距和建议 |
| 定制简历生成页 | `resume-generation.png` | 展示 Markdown 简历预览和修改原因 |
| 真实性风险检测页 | `truth-check.png` | 展示风险项和保守表达提醒 |
| 面试追问预测页 | `interview-questions.png` | 展示追问问题、原因、建议回答和风险提醒 |
| Markdown / DOCX 导出 | `export-actions.png` | 展示复制、Markdown 导出、DOCX 导出和模板选择 |
| 用量统计页 | `usage.png` | 展示 AI 调用次数、额度和最近记录 |
| 个人中心页 | `account.png` | 展示账户信息、昵称修改和用量概览 |
| 管理后台页 | `admin.png` | 展示用户列表、状态管理和全站用量 |

建议截图尺寸：

- 桌面端：`1440 x 900` 或接近 16:10。
- 浏览器缩放：100%。
- 数据：使用演示账号和脱敏示例内容，不要截图真实简历、真实手机号、真实邮箱或真实 API Key。
