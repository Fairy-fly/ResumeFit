# ResumeFit V0.8 DOCX 模板增强版验收清单

## 一、V0.8 已完成内容

- [ ] DOCX 导出接口支持 `template` query 参数。
- [ ] 支持 `standard`、`modern`、`compact` 三种模板。
- [ ] 不传 `template` 时默认 `standard`。
- [ ] 非法模板参数返回 `422`。
- [ ] 前端 `/versions` 页面新增 DOCX 模板下拉选择。
- [ ] “导出 DOCX”按钮按当前选中模板下载文件。
- [ ] Markdown 导出继续保留。
- [ ] DOCX 模板导出不调用 AI。
- [ ] DOCX 模板导出不计入 AI 使用额度。
- [ ] DOCX 模板导出沿用用户隔离。
- [ ] 未新增数据库表。
- [ ] 未实现 PDF、在线 Word 预览、复杂模板编辑器或导出历史。

## 二、后端接口验收

### 1. 默认模板

操作步骤：

1. 登录用户 A。
2. 准备用户 A 自己的 ResumeVersion。
3. 调用 `GET /resume-versions/{id}/export/docx`。

预期结果：

- [ ] 返回 `200`。
- [ ] `Content-Type` 为 `application/vnd.openxmlformats-officedocument.wordprocessingml.document`。
- [ ] `Content-Disposition` 包含 `.docx`。
- [ ] 文件名包含 `standard` 或等价默认模板标识。
- [ ] DOCX 可被 Word 或 `python-docx` 打开。
- [ ] DOCX 内容包含简历核心文本。

### 2. standard 模板

操作步骤：

1. 调用 `GET /resume-versions/{id}/export/docx?template=standard`。

预期结果：

- [ ] 返回 `200`。
- [ ] 下载文件非空。
- [ ] 内容来自当前 ResumeVersion 的 Markdown。

### 3. modern 模板

操作步骤：

1. 调用 `GET /resume-versions/{id}/export/docx?template=modern`。

预期结果：

- [ ] 返回 `200`。
- [ ] 下载文件非空。
- [ ] DOCX 可被读取。
- [ ] 内容来自当前 ResumeVersion 的 Markdown。

### 4. compact 模板

操作步骤：

1. 调用 `GET /resume-versions/{id}/export/docx?template=compact`。

预期结果：

- [ ] 返回 `200`。
- [ ] 下载文件非空。
- [ ] DOCX 可被读取。
- [ ] 内容来自当前 ResumeVersion 的 Markdown。

### 5. 非法模板

操作步骤：

1. 调用 `GET /resume-versions/{id}/export/docx?template=bad`。

预期结果：

- [ ] 返回 `422`。
- [ ] 不生成业务数据。
- [ ] 不调用 AI。

## 三、权限与安全验收

操作步骤：

1. 未登录访问 `GET /resume-versions/{id}/export/docx?template=modern`。
2. 用户 B 尝试导出用户 A 的 ResumeVersion。
3. 检查导出文件名。

预期结果：

- [ ] 未登录返回 `401`。
- [ ] 用户 B 导出用户 A 的版本返回 `404`。
- [ ] 文件名不包含 token。
- [ ] 文件名不包含 email。
- [ ] 文件名不包含用户 id。
- [ ] 导出文件不保存到公开目录。
- [ ] admin 默认也不能绕过用户隐私导出他人简历。

## 四、前端页面验收

操作步骤：

1. 登录用户。
2. 打开 `/versions`。
3. 选择一个历史 ResumeVersion。
4. 查看 DOCX 模板下拉框。
5. 选择 `Standard` 并导出 DOCX。
6. 选择 `Modern` 并导出 DOCX。
7. 选择 `Compact` 并导出 DOCX。

预期结果：

- [ ] `/versions` 页面正常打开。
- [ ] 模板下拉框默认选中 `Standard`。
- [ ] 选择 `Modern` 后请求带 `template=modern`。
- [ ] 选择 `Compact` 后请求带 `template=compact`。
- [ ] 请求携带 `Authorization: Bearer <token>`。
- [ ] token 没有拼接到 URL。
- [ ] 导出时按钮有 loading 状态。
- [ ] 导出中不能重复点击。
- [ ] 导出失败显示友好错误。
